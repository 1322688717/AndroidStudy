# 中文 Android jetpach 教程

## 简介

* 谷歌推荐的一份使用的开发工具

* jetpach分为四个领域

  * 架构
    * 分页
    * 导航
    * 工作管理器
  * UI
    * 动画
    * 转场
    * 布局
    * ui组件
  * 行为
    * 通知
    * 许可
    * 分享
  * 基础
    * appcompat
    * texting libraries

  #### Room

  * 简单的实体类创建
    * `data class User(`
    * `val id: String,`
    * `val userName : String,`
    * `……)`
